📋 How to Use

  1. Local Storage: All documents are saved locally in your browser
  2. Document Types: Support for 7 different document types (RC, BOL, POD, etc.)
  3. File Upload: Upload PDF, PNG, JPG, and JPEG files up to 10MB
  4. Search & Filter: Find documents quickly by type, date, or keywords
  5. Optional Cloud Sync: Use Cloud Dashboard to sync across devices (optional)
  6. Mobile Friendly: Works on all devices with responsive design
