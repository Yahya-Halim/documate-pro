🔧 Setup Instructions

  1. GitHub Token Required: Click "Cloud Dashboard" and setup your GitHub Personal Access Token
  2. Automatic Sync: All changes are automatically saved to GitHub
  3. Cross-Device Access: Access your data from any device with internet
  4. Offline Mode: If GitHub is unavailable, data is saved locally
  5. Data Security: Your data is stored in your GitHub repository
